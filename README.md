# Sakaban-qqBot

## 项目介绍
一个基于melobot的qqbot，用于好友群聊骚

## 项目使用
版本：python>=3.10,我用的3.10
所需库安装：见requirements
运行flask_img.py


要开sd的话把config里的enable改成true
把对外开放的url填入config
把spark3.5的id，key，secret填入config(不开sd就不用填了)
sd可以用秋叶包开下api，原生webui也行

运行demo.py
